# todo-app-react-redux

How to Use

1.Install Dependencies:
  npm install
2.Run the App:
  npm run dev

Features
  Add new todos.
  Remove todos.
  Mark todos as completed/incomplete.
  Todos are saved to local storage, allowing them to persist even after the page is refreshed.
  
Technologies Used
  React
  Tailwind CSS
  Redux
  Local Storage
